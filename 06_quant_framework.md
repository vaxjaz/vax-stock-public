**v1.4 三个关键发现(修复后)**：
1. **核心因子 IC/ICIR 不降反升**：净利同比 0.0201→0.0240、户数 -0.0174→-0.0219、合成评分 0.0132→0.0202。原因——被剔除的退市/调出股恰是因子的正确反面样本(业绩差→股价跌→被调出),加回后因子更有效。先验"业绩因子会降"的预期是错的。
2. **IC ≠ 收益**：净利同比 IC 最高(0.024)但无多空验证;**主力净流入 IC 仅第2,但多空夏普0.90/年化12.89%/回撤最小,实战最强**。定权重应参考"多空夏普 + 分位单调性",不能只看 IC。
3. **合成评分适合选股不适合对冲**：分位 Q0→Q4 完美单调递增,但斜率平→多空夏普仅0.07、回撤-24.93%。它的价值在多头排序,本就如此用。

⚠️ **权重精确数值的调整(如提主力净流入权重)待 harness + 样本外验证后再动**(见 `11_validation_roadmap.md` 第1步);本表确立的是**方向**(强者:主力净流入/净利同比/合成评分/户数;清零:换手Z/PE分位/20日位置/RSI/MA5/资金流斜率)。

## 二、右侧信号评分模型 v1.2

### 评分阈值

```
总分 ≥ 3.5 → 强买入信号
总分 ≥ 2.0 → 可考虑介入
总分 ≥ 0.5 → 观察等待
总分 < 0.5 → 回避
```

### 核心因子评分规则

#### ① 10日主力净流入（v1.2市值归一化分档）

**v1.1的问题**：10日流入>0统一+1.5分。长江电力流入0.2亿（占6800亿市值的0.003%）和风华高科流入0.2亿（占380亿市值的0.05%）同分——但含义差20倍。

**v1.2解决方案**：按`10日净流入 / 流通市值`的百分比分档。

| 条件 | 评分 | 含义 |
|---|---|---|
| ratio ≥ 0.5% | **+1.5** | 相对市值的强流入 |
| ratio ≥ 0.1% | +1.0 | 中等流入 |
| ratio > 0 | +0.5 | 弱流入 |
| ratio ≤ -0.5% | -1.0 | 相对市值的重度流出 |
| 其他 | 0 | 中性 |
| 无市值数据 | 回退v1.1二值逻辑 | 兼容旧数据 |

**实测效果（2026-06-11数据）**：

| 股票 | v1.1评分 | v1.2评分 | 原因 |
|---|---|---|---|
| 生益科技 | 3.0 | **2.0** | +0.04亿对350亿=0.01%，弱流入降级 |
| 三环集团 | 2.5 | **1.5** | +0.24亿对1250亿=0.02%，微量流入 |
| 中际旭创 | 2.5 | **3.0** | -7.5亿对11000亿=-0.07%，中性撤销惩罚 |
| 工业富联 | 2.0 | **2.5** | -3.24亿对13800亿=-0.02%，中性而非重度流出 |

注意中际旭创和工业富联**反而升分**——v1.1错罚了大盘股，几亿流出对万亿市值确实是噪音。

⚠️ **分档阈值(0.5%/0.1%)为逻辑设定，待9月回测验证后校准。**

#### ② 净利同比（与v1.1一致）

| 条件 | 评分 |
|---|---|
| > 50% | +1.5 |
| > 20% | +1.0 |
| > 0% | +0.3 |
| < -20% | -0.5 |

#### ③ 股东户数变化（与v1.1一致）

| 条件 | 评分 |
|---|---|
| < -2%（强集中） | +1.0 |
| -2%~0%（轻微集中） | +0.5 |
| > +10%（强分散） | -0.5 |

### 辅助因子

| 条件 | 评分 | 说明 |
|---|---|---|
| 收盘价低于MA5超过3% | +0.5 | ⚠️方向反转：低吸优于追涨 |
| 换手Z-score > 2 | -1.0 | 异常放量警惕派发 |

### 市场环境过滤（v1.2含恐慌豁免）

#### 动量市（momentum）
- 反转因子（PE/20日位置/RSI）不参与评分
- 只看资金流+业绩+户数

#### 价值市（value）
- PE历史百分位<30 → +0.5
- 20日位置<30 → +0.5

#### 恐慌市（panic）— v1.2新增豁免机制

**v1.1的问题**：恐慌市一律-1.0分，所有股票都被打成"回避"。但2026-06-10恐慌日（跌停54只），恒瑞医药当天PE历史1%+业绩+22%+20日位置15%——这恰恰是历史最佳买点，v1.1却系统性回避了。

**v1.2解决方案**：三重门槛豁免

```
如果同时满足以下三个条件，免除恐慌惩罚(-1.0)：
  ① 净利同比 > 20%      ← 业绩确认
  ② PE历史百分位 < 20%   ← 估值确认
  ③ 20日位置 < 30%       ← 位置确认

满足：标记"💎恐慌市优质低位"，惩罚归零
不满足：正常扣-1.0分
```

**实测效果**：

| 场景 | v1.1 | v1.2 |
|---|---|---|
| 恐慌市+恒瑞（业绩+22%/PE 1%/位置15%） | 0.0（被扣1.0） | **1.0+💎**（豁免） |
| 恐慌市+科华（业绩+13%不达20%阈值） | -0.7 | **-0.7**（不豁免） |

三重门槛防止在恐慌市中接刀——只有业绩+估值+位置三重确认的真优质股才能豁免。

## 三、市场环境分类器(短期 regime)v1.2(含状态平滑)

> v1.3 起,本章节描述的是**短期 regime**(1-2周时间窗口的市场风格判断),
> 与"第八章 宏观环境与 Macro Regime"中的**宏观 regime**(1-3个月环境)叠加使用。
> 短期 regime 不变,v1.2 逻辑完整保留。

### v1.1的问题

创业板单日跑赢上证2%就切换到momentum——6月10日恐慌（panic），6月11日又回momentum，单日切换导致评分跳变。

### v1.2解决方案：状态平滑

新增`regime_history.json`状态持久化文件，记录每日原始信号并平滑切换：

| 切换场景 | 规则 |
|---|---|
| momentum ↔ value | **连续2日同向原始信号**才切换 |
| 任何状态 → panic | **单日立即生效**（安全优先） |
| panic → 解除 | **连续2日非panic信号**才解除 |

### 判断标准（与v1.1一致，平滑逻辑为v1.2新增）

| 市场状态 | 原始信号判断 | v1.2平滑规则 |
|---|---|---|
| **动量市** | 创业板/科创平均涨幅 - 上证涨幅 ≥ 2% | 需连续2日确认 |
| **价值市** | 上证涨幅 - 创业板/科创平均涨幅 ≥ 1% | 需连续2日确认 |
| **恐慌市** | 跌停数 > 50只 | 单日立即，解除需2日 |
| **默认** | 以上均不满足 | 动量市（A股近年偏成长） |

### 实测效果（5场景测试）

```
Day1 动量信号       → momentum ✅
Day2 单日价值信号    → momentum ✅（平滑抑制噪音）
Day3 连续2日价值信号 → value    ✅（确认切换）
Day4 跌停80只       → panic    ✅（立即生效）
Day5 恐慌解除第1日   → panic    ✅（解除需2日确认）
```

## 四、经典识别模型（v1.2 修正）

### 真便宜（可介入）

```
v1.2: 10日主力净流入占市值≥0.1% + 净利同比>20% + 股东户数集中
      (去除52周位置/PE历史低条件——动量市失效)
案例: 中际旭创（户数-15.78%+净利+262%+v1.2评分3.0）
```

### 真危险（回避）

```
v1.2: 10日累计净流出占市值≤-0.5% + 换手Z-score>2 + 股东户数分散
案例: 国瓷材料（10日流出+换手17%+高位）
```

### 价值陷阱

```
股价跌 + PE历史百分位高 + 净利同比大幅恶化
案例: 比亚迪（PE历史78% + 净利-55%）
```

### 派发陷阱

```
股价大涨 + 换手Z-score异常 + 股东户数增加 + 10日累计净流出
案例: 沪电股份(+6.93%但主力流出1.19亿 + 户数+38.87%)
```

### 恐慌市优质低位（v1.2新增）

```
恐慌市(跌停>50) + 净利>20% + PE历史<20% + 20日位置<30%
→ 三重确认后豁免恐慌惩罚，标记"💎恐慌市优质低位"
案例: 恒瑞医药（2026-06-10恐慌日，业绩+22%/PE 1%/位置15%）
```

## 八、宏观环境与 Macro Regime (v1.3 新增)

### 8.1 设计动机

v1.2 之前的框架专注于**个股+板块+短期 regime** 三个层次,缺乏对**宏观环境**的系统性观察。在某些极端情况(2024年初的雪球敲入、2024年9月的政策底等),仅靠短期 regime 无法捕捉到大环境的转折。v1.3 引入 6 个宏观维度,形成"短期 × 宏观"双层 regime 决策框架。

**关键原则**:宏观层指标**仅用于仓位决策**,不参与个股评分公式。这是因为宏观信号月度更新慢,对个股选股的边际信息含量低,但对仓位调整极有价值。

### 8.4 综合 Macro Regime 判定

```python
def combine_signals(signals: List[str]) -> str:
    bull = signals.count("✅") + 2 * signals.count("✅✅")
    bear = signals.count("❌") + 2 * signals.count("❌❌")
    if bull >= 4 and bear <= 1:  return "🟢 强看多"
    if bull >= 3 and bear <= 2:  return "🟢 看多"
    if bear >= 4 and bull <= 1:  return "🔴 强看空"
    if bear >= 3 and bull <= 2:  return "🔴 看空"
    return "🟡 中性"
```

### 8.5 短期 × 宏观 双层 Regime 叠加矩阵(核心决策表)

| 短期 v1.2 | 宏观 v1.3 | 应对策略 | 仓位建议 |
|---|---|---|---|
| momentum | 🟢 看多 | 强势上涨期 | 加仓 60-80% |
| momentum | 🟡 中性 | 谨慎跟随 | 维持 40-60% |
| momentum | 🔴 看空 | **顶部警告** | 降至 30-40% |
| value | 🟢 看多 | **价值底确认** | 加仓 70-90% |
| value | 🟡 中性 | 选股优先 | 维持 40-60% |
| value | 🔴 看空 | 价值陷阱多 | 维持 30-40% |
| panic | 🟢 看多 | **黄金底** | 大胆加仓 80-100% |
| panic | 🟡 中性 | 逐步建仓 | 加仓 50-70% |
| panic | 🔴 看空 | 双杀,坚守现金 | **降至 20% 以下** |

**关键解读**:
- `panic + 🟢看多` = 历史级买点。短期市场恐慌但宏观环境支撑,这是2018年底/2024年9月级别的机会
- `momentum + 🔴看空` = 牛市末段。短期还在涨,但宏观流动性已转向,警惕回撤
- `value + 🟢看多` = 价值底确认。低估值标的有宏观环境助攻,胜率最高
