# 第十章 分层入场标准 (v1.4 草案)

> 起草于 2026-06-15。动机:v1.3 的入场逻辑对所有标的套用同一把"右侧确认"尺子,
> 导致深度低估优质股(如恒瑞)被误用动量股标准,出手频率畸低。
> 本章拆分为**三条独立入场通道**,各有适配的标的类型与门槛。
>
> ⚠️ **本章所有数值阈值均为逻辑设定,标注"⚠️待9月回测"者优先验证。**
> ⚠️ **核心原则:放松出手频率,不放松出手质量。抗派发保护一律不动。**
 
---

## 10.1 设计哲学:为什么要分层

v1.3 之前,任一候选都要过"评分≥2.0 + 不追高 + 右侧确认(三选二)"三层**相乘**过滤。
三层每层都合理,但相乘后任何单日通过者≈0(2026-06-15 实测:40 只观察池 → 评分≥2.0 仅 6 只 → 去高位剩 3 只 → 过右侧确认 ≈0)。

**问题不在评分系统(开火正常 6/40),而在于用动量股的右侧标准去卡所有标的。**

解决:按标的**性质**先分流,再用适配门槛:

```
候选标的
   │
   ├─ 深度低估质地(估值/位置极低 + 业绩正)──→ 【通道A 左侧建仓】不要求趋势反转
   │
   ├─ 动量/再入场标的(评分≥2.0 + 业绩驱动)──→ 【通道B 右侧追入】要求确认(防派发)
   │
   └─ 短线题材龙头(板块启动 / 涨停板)──────→ 【通道C 机会仓】独立止损,门槛放宽
```
 
---

## 10.2 通道A:左侧建仓(深度低估质地)

**适配标的**:估值与位置已到历史极低、业绩正向的优质防御/价值股。
**核心松绑**:**不要求**"放量站均线"的右侧反转确认 —— 这类标的的安全垫来自估值与位置,而非趋势。
**底线守住**:左侧 ≠ 接刀,**不在放量急跌中买入**。

### 入场门槛(常态市,需全部满足)

| 维度 | 条件 | 依据 |
|---|---|---|
| 估值 | PE 历史百分位 **< 10%** | 安全垫核心 |
| 位置 | 52周位置 **< 15%** | 深度低位 |
| 基本面 | 净利同比 **> 0**(理想 >20%) | 排除价值陷阱 |
| 资金 | 10日主力**不重度流出**(ratio > -0.1%),最好转流入 | 排除资金弃守 |
| 止跌 | **非**放量急跌、**未**放量破前低 | ⚠️不接刀 |

### 恐慌市子情形(复用第二章豁免)

恐慌市(跌停>50)中,改用既有"优质低位三重豁免"阈值,可更早左侧:
- 净利同比 >20% + PE历史百分位 <20% + 20日位置 <30% → 标记 💎,立即可分批
### 执行方式

| 参数 | 规则 |
|---|---|
| 建仓节奏 | **分批**(建议3批),不一次满仓 |
| 加仓纪律 | 跌不加(除非放量反转确认),企稳/回踩不破再补 |
| 止损 | 估值保护强,以**基本面恶化**(净利转负/PE分位快速抬升)为退出,而非单纯价格止损 |

### 当前适用案例

**恒瑞医药 600276**:PE 历史 **3%** + 52周 **9%** + 净利 **+22%** + 5日资金转流入(+0.15亿)+ 今日-1.26%窄幅非急跌 → **符合通道A,可左侧分批**。此前误用动量股右侧标准要求其"放量站MA10",是 v1.3 的尺子错配。

## 10.3 通道B:右侧追入(动量/再入场标的)

**适配标的**:业绩驱动、评分≥2.0、趋势型的动量股或主升浪再入场。
**为什么严**:动量股高位易接到**派发**(沪电股份案例),右侧确认是抗派发安全锁,**不放松**。

### 入场门槛(需全部满足)

| 层级 | 条件 |
|---|---|
| 评分 | right_side_score **≥ 2.0**(⚠️**不下调**,1.5档多含流出票) |
| 右侧确认 | **三选二**(⚠️**不砍为一选一**):①放量阳线收复关键位(MA10/MA20)②连续2日主力净流入 ③板块同步走强/企稳 |
| 派发否决 | 🚨 换手Z>2 + 户数分散 + 10日净流出 三者齐 → **直接否决** |

### 高位判定:单维度 → 集群(本次松绑点)

**v1.3 问题**:单看 20日位置高 或 PE分位高 即否决。但第一章实证:**20日位置/PE分位在动量市是失效因子**,单维度否决与框架自相矛盾。

**v1.4 改为两级判定**(2026-06-15 实盘修订)。

> ⚠️ **修订背景**:初版把"资金流出"设为否决的**必要条件**,实盘两处误触发——
> 生益(位置103%/PE100%但今日微流入)漏放、中际(高位但筹码强集中+今日特大单流入)误杀。
> 故改为"位置×估值极端度"为主、"今日特大单方向"为辅的两级规则,不再让单日资金流向单点决定。

```
═══════════════════════════════════════════════════════
第一级 绝对极端 → 硬否决(任何筹码/资金信号不豁免)
═══════════════════════════════════════════════════════
触发: 52周位置 ≥ 95%  AND  PE历史分位 ≥ 95%
理由: 位置与估值同处历史顶端5%,无论谁在买,接棒风险压倒一切
 
═══════════════════════════════════════════════════════
第二级 高位非极端(未达第一级,但 52周>85% 或 20日>90%)
═══════════════════════════════════════════════════════
看"今日特大单方向 + 筹码趋势"二选一定性:
  • 今日特大单净流出 且 (10日资金流出 或 户数分散) → 否决
  • 今日特大单净流入 或 户数强集中(<-10%)            → 降研究层(智能资金在吸,跟踪不追)
 
═══════════════════════════════════════════════════════
均不满足上述 → 正常进入通道B(右侧确认)
═══════════════════════════════════════════════════════
```

**关键**:第一级的"52周≥95%"是硬门槛——它把中际(94%)挡在第一级之外、落入第二级,
而第二级再凭"今日特大单净流入"把中际判为研究层而非拉黑。生益/风华(均≥95%/≥95%)直接第一级否决。

### 当前适用案例对比

| 标的 | 评分 | 位置/PE分位 | 今日特大单 | 处理 | 触发级 |
|---|---|---|---|---|---|
| 工业富联 601138 | 2.0 | 44%(20日)/36% | -0.02亿 | 通道B,待右侧确认 | 未达高位,正常进通道B |
| 中际旭创 300308 | 2.5 | 52周94%/96% | **+0.89亿净流入** | **降研究层**(跟踪不追) | 第二级:52周94%<95%落入二级,特大单净流入→研究层 |
| 北方华创 002371 | 1.8 | 52周90%/98% | +0.98亿净流入 | 降研究层 | 第二级:特大单净流入,但业绩仅+3%质地一般 |
| 生益科技 600183 | 2.0 | 52周103%/**100%** | +0.20亿 | **硬否决** | 第一级:≥95%且≥95%,微流入不豁免 |
| 风华高科 000636 | 2.0 | 52周97%/**97%** | -0.25亿净流出 | **硬否决** | 第一级:≥95%且≥95% |
| 鼎龙股份 300054 | 0.5 | 52周95%/**99%** | +0.07亿 | **硬否决** | 第一级:≥95%且≥95%(叠换手Z2.85异常) |

## 10.4 通道C:机会仓(短线题材,独立止损)

**为什么是最该放松的点**:单次仅出 5%、独立止损、与主仓隔离 —— **下行被结构性锁死**,是全系统风险最有界的通道。机会仓信号长期空转(2026-06 多日 0 候选),是阈值卡太死,而非市场无机会。

### 信号B 阈值调整(板块启动初期低位龙头)

| 条件 | v1.3 | v1.4 草案 | 标记 |
|---|---|---|--|
| 板块今日涨幅 | ≥2% | ≥2%(不变) | |
| 板块3日累计净流入 | ≥5亿 | **≥3亿** |  |
| 个股20日位置 | <60% | **<70%** |  |
| v1.2评分 | ≥1.5 | ≥1.5(不变) | |
| 净利同比 | >0 | >0(不变) | |
| 流通市值 | ≥30亿 | ≥30亿(不变) | |
| 止损 | -5% | -5%(不变) | 🔒安全锁不动 |

**对应脚本常量(`opportunity_scanner.py`)** —— 文档与代码可追溯对齐:

| 改动 | 行 | 类型 | 5亿/60% → 3亿/70% |
|---|---|---|---|
| `B_SECTOR_MIN_3D_INFLOW` | 69 | 🔧逻辑 | `5.0` → `3.0` |
| `B_STOCK_MAX_POS_20D` | 71 | 🔧逻辑 | `60` → `70` |
| docstring | 522 | 📝文字 | `> 5亿` → `> 3亿` |
| logger | 578 | 📝文字 | `>5亿的板块` → `>3亿的板块` |
| 报告输出行 | 842 | 📝文字 | `3日流入>5亿 + 位置<60%` → `>3亿 + <70%` |

⚠️ 逻辑与文字须同步:只改 69/71 不改 842,报告会出现"按3亿筛选却提示5亿"的口径矛盾。

### 信号A(涨停板次日追板)

阈值**不变**。其"逆向选择"(一字板买不进,买到的是封单弱的)是机制性缺陷,放松阈值无法解决,只能靠 -3%/开板即走的止损纪律控制单次损失。

### 仓位规则(不变)

总额 15%(~24600元)/ 单次 5%(~8200元)/ 最多同时 3 只 / 亏损不影响主仓。

## 10.5 仓位补位(执行既有矩阵,非放松)

当前主仓约 **33%**,低于第八章叠加矩阵"动量市 × 🟡中性 = 40–60%"的**下沿**。
→ 用通道A最干净的左侧票(当前:恒瑞)**补至 40%**。
这是**执行你自己定的矩阵**,不是放松门槛。
 
---

## 10.6 守住不调的项(纪律红线,等9月回测再议)

| 项 | 维持 | 理由 |
|---|---|---|
| 评分门槛 2.0 | ❌不下调 | 评分被业绩主导,1.5档当前全是大额流出票,降门槛=放进派发风险 |
| 右侧"三选二" | ❌不砍 | 其中"连续2日资金确认"是抗派发核心锁,砍掉=拆安全锁(沪电反例) |
| 派发三件套否决 | ❌不动 | 换手Z>2+户数分散+流出,任何通道都一票否决 |
| 机会仓-5%/-3%止损 | 🔒锁死 | 放宽入场的前提正是止损不松 |
| 左侧"不接刀" | 🔒锁死 | 放量急跌中买入不属左侧,属接刀 |

## 10.7 净效果(2026-06-15 数据重跑)

| | v1.3(过紧) | v1.4草案 |
|---|---|---|
| 左侧通道 | 无(误卡恒瑞) | **开恒瑞**,补仓位至40% |
| 右侧研究层 | 中际旭创被拉黑 | 中际进研究层(仍不追) |
| 机会仓信号B | 0候选 | 阈值放宽后可能出1–2候选 |
| 高位真危险 | 生益/风华 | **维持否决**(集群成立) |

从"几乎0买点"→"2–3个分层入场机会",**且无一为追高位/追流出**。抗派发保护一处未动。
